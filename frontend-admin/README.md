## Веб‑админка

- Интерфейс для администраторов и методистов.
- Управление вузами, ГОСТами/стилями, шаблонами, правилами, автоисправлениями, продуктами и платежами, контентом бота.

Рекомендуемый стек:

- React + TypeScript,
- UI‑библиотека (например, Headless UI) + собственная дизайн‑система (white + light blue, liquid glass),
- локальный набор Lucide‑иконок (SVG).
