## Клиентский веб‑апп (Telegram Mini App)

- Mobile‑first интерфейс для студентов.
- Авторизация через initData Telegram (будет интегрирована с `/api/miniapp/auth/telegram`).
- Основные разделы: Главная, Демо, Проверка, История, Профиль.

Технологический стек (предложение):

- React + TypeScript (например, Vite),
- TailwindCSS или CSS‑модули с кастомным дизайном (liquid glass, iOS‑style),
- локальный набор иконок Lucide (SVG) через собственный скрипт/компонент.
